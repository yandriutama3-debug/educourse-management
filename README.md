# EduCourse Management System

## Deskripsi
Aplikasi manajemen akademik kampus (tanpa database).

## Fitur
- Routing Dinamis
- MVC Laravel
- Blade Component & Layout
- Bootstrap 5 & Dark Mode JS

## Cara Menjalankan
1. `composer install`
2. `php artisan serve`

## Teknologi
Laravel 10, Bootstrap 5
