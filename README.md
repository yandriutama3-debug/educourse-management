# EduCourse Management System

## Deskripsi
Aplikasi manajemen akademik berbasis web menggunakan Laravel 10 (tanpa database). Digunakan untuk mengelola simulasi data mahasiswa, dosen, mata kuliah, serta menampilkan dashboard dan profil.

## Fitur
- Routing Dinamis & Lengkap (Parameter, Optional, Group, Fallback)
- MVC Architecture (Controller & View Blade)
- Bootstrap 5 Modern UI (Card, Modal, Accordion, Badge, dll)
- Responsive & Mobile Friendly
- Dark Mode Toggle
- Active Navbar State
- Dummy Data Implementation (Arrays)

## Cara Install & Menjalankan Project
1. Clone repository ini: `git clone [link-repo-anda]`
2. Masuk ke folder project: `cd educourse-management`
3. Install dependencies: `composer install`
4. Copy file environment: `cp .env.example .env`
5. Generate app key: `php artisan key:generate`
6. Jalankan server lokal: `php artisan serve`

## Teknologi
- Framework: Laravel 10
- Frontend: Bootstrap 5, Blade Template Engine
